Project Video Requirements

You must post a video demonstrating the finished project in the homework page (this video does NOT count as the content video required for objective 3). The video must accomplish these tasks:

  1. Describe how the finished project meets the project requirements, including:
    1. how the finished project meets the needs of the identified target audience
    2. the site responsiveness on a variety of screens
    3. the validity of the html and css used in the site
    4. the use of JavaScript and/or jQuery together with JSON to provide data to site pages
    5. the usability of the site
    6. the use of the PARC and typography design principles in the site
  2. How the project fulfills your approved proposal
  3. How you addressed issues raised in the peer review(s)
