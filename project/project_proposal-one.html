Project Proposal I

The first step in creating your project proposal is to choose a subject for your site.You should review the requirements for the project as you consider your topic. Remember especially that you will need to pull some of the information from your site from an external JSON file.

Once you have thought of a subject write down the 5 pages you will make with a few details about the content each of those pages will hold. If you have a hard time coming up with 5 pages with good content, you may want to reconsider your topic. If you have a hard time keeping it to only 5 pages...you might also want to reconsider...;)

Tasks

  1. Create a new html file to become the start of your proposal. Add to it the critical HTML and semantic structural elements.
  2. Add a section describing your subject and the purpose of the site. Be specific!
  3. Add a second section with the list of 5 content pages along with a short description of what will be on each page.
  4. Validate your page and add a link to the proposal to your Temporary Homepage.
  5. Submit the URL to your site in Ilearn.

Grading Matrix

This assessment will be graded for two objectives: 1 and 4.

Objective 1 - 18 points

  1. The page is valid HTML5 - 3 points
  2. The page has a descriptive title and description - 3 points
  3. The page has semantic header, nav, main and footer elements - 3 points
  4. All images load correctly - 3 points
  5. There is a working link back to the Temporary home page - 3 points
  6. The page is named following the web naming rules - 3 points

Objective 4 - 8 points

  1. The purpose statement clearly describes why the site is being built - 3 points.
  2. Five content pages are listed with examples of the content that will be on each - 5 points
