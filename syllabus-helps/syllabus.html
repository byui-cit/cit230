CIT 230 - Web Frontend Development

This course focuses on HTML, CSS, JavaScript and other web development basics. There is a lot to learn quickly. If you are new to web development, especially HTML and CSS, you might consider taking WDD 100 to help you prepare for this course. WDD 100 is not required for CIT 230, but it will help you be prepared. In addition, about 2/3 of students have had or are taking a programming course (CIT 160) - this seems to make a difference in their success in this course, particularly when working with JavaScript.

This is a technical class, not a design class. It will require between 9-12 hours per week. Even though it may be challenging, you will be successful as long as you are diligent and work hard. You are required to collaborate with your classmates in order to support and help one another.

Purpose

When building web sites, successful web developers must consider all of the following: audience, organization, interface, usability, content, layout, and visual (aesthetic) design. This course will teach you to plan, design, and construct web sites to work properly in modern web browsers and conform to web standards while following best practices.

Keys to Success

To be successful in this course you will need to have the following:

  • Strong work ethic
  • Ability to learn and apply what you learn
  • Ability to solve problems
  • Ability to meet deadlines
  • Ability to work well with others — supporting, communicating, teaching, and learning from them

Course Objectives

These objectives are based on web standards and best practices to help you be successful in the future.

  1. Develop web pages that are semantic and valid using the HTML5 markup language.
  2. Style web pages with valid Cascading Style Sheets (CSS) level 3 and avoid all HTML presentational markup.
  3. Provide content that is visually appropriate, usable, and findable to humans and machines.
  4. Plan, design, and develop web pages and sites according to best practices of organization and maintainability.
  5. Work with JavaScript to handle events and interactions within the browser including JSON and remote API's.
  6. Work effectively with others in a work environment by communicating clearly, fulfilling assignments and meeting deadlines.

Frameworks and Templates

Note: This is a web course intended to help you learn how to construct web sites using HTML5, CSS and JavaScript. Outside templates are not allowed (except as introduced in the course). Pages built from site builder software, frameworks, templates or drag-n-drop tools are NOT allowed.

Know, Do, Become

This course is the first step to becoming a web developer - which requires you to both know and do. In order to know something, you must study, reflect, and practice. To do requires effort and application to implement your knowledge. If you consistently do this you are on your way to becoming what you desire.

Collaboration

During this class, you will form and work with learning teams. These teams will be a primary resource for mutual support and feedback. Before you turn anything in, get feedback from your team.  This kind of team structure is similar to what you will find in the workplace.

You will start organizing your team during week 01 while you are introducing yourself to your classmates. Look for others you can collaborate with. It is vital that you coordinate schedules and identify times that you are available to meet with your peers.

Assessments

Each class assessment relates to one or more course objectives, indicated in brackets. Due dates are listed in the course schedule. Each assessment will be submitted as specified in its description. Once posted, an assessment should remain unchanged until graded, with the grade appearing in the grade book. Each assessment will be graded using the provided grading matrix. It is YOUR responsibility to thoroughly read and understand the specification for each assessment and test your own product for compliance to the stated criteria. Peer review, feedback and correction using the grading matrix prior to the due date is encouraged.

Weekly Assessments [Varied]

Each week you will have a variety of assessments, including quizzes, to allow you to solve problems and apply the concepts discussed in the class. Each assessment includes its own grading matrix which should be carefully reviewed prior to submission. Team collaboration is encouraged while completing each assessment.

Web Site Project [All Objectives]

The Web Site Project will be your major project for the course. It will require an implementation of all the skills and technologies covered by the course. This is a major aspect of the course and will require substantial time and effort. Team collaboration is encouraged while completing the site project.

Getting Help

For many students learning HTML, CSS and many of the tools and techniques of this course is equivalent to learning a foreign language. For 99% of students, it is not a matter of "if", but rather "when" you will need help. There are a number of mechanisms built into the course so you can get help. They are:

  • Learning Teams — This is a small group of your peers, enrolled in the same class, doing the same preparations, activities and assessments. You should interact with this group frequently — ask questions, share insights, read one another’s work, make comments and suggestions. Learn how to work together and support one another. This is a very powerful and useful model — one that is used in the web design and development industry. 
  • Weekly Discussion Board — where you can ask and answer questions, ask for clarification and share insights (collaborate) with your peers. This is an avenue to maintain communications outside of class. The professor and the teaching assistants will monitor the board and offer help when appropriate. However, the primary intent is for class members to help one another using this tool. Don’t underestimate the importance of these activities to your learning.
  • The Professor — Feel free to make an appointment to visit with me at calendly.com/thompsonda or visit with me in class.
  • Teaching Assistant — A TA is available in-class with whom you can visit and get help.
  • CIT Web Lab — STC 347 is open for a limited number of hours each week with a TA available to help on a first come, first serve basis. The lab hours will be made available usually during the second week of the semester.
  • Tutoring Center — The BYU-Idaho tutoring center hires tutors to meet and assist students. Visit the tutoring center web site to check availability and schedule an appointment.
