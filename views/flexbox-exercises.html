Flexbox Exercises

Overview

"Practice makes perfect." You've probably been told this at some point in your life...usually when you didn't want to do something if you are anything like me ;) But the truth of the matter is that with many things...most things...the best way to master them is by practicing.

Tasks

Complete the following with your learning group:

  1. Complete at least three of these exercises using flexbox.
  2. Try to complete all levels of Flexbox Froggy.
  3. Earlier in the semester you were asked to convert this wireframe into HTML. If you did not complete that then...you will need to do that now. With that HTML file now add the CSS necessary to make the page match the actual layout of the wireframe.
  4. As you finish the items above show your work to either the instructor or TA in class to receive credit.

Grading

  • Flexbox exercises complete. - 5 points
  • Flexboggy Froggy. - 5 points
  • Wireframe layout correctly built. - 10 points
