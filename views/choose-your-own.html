Choose your Own Adventure Activity

Overview

When I was younger Choose Your Own Adventure books were very popular. The story would be laid out in branching paths...some led to success, others to failure (usually a gruesome death). But they were always entertaining. This activity will have you build a simple engine that could be used to create your own adventure story.

Tasks

Begin by grabbing the starter code below

See the Pen yXMEoE by Shane T (@matkat) on CodePen.

You can either create a new html and JS file in your sandbox, or click the "Edit in Codepen" link above and fork it. This option would require a free Codepen account.

Spend a few minutes looking over the code. You have been given a JSON object that contains the structure of our story. There is also a function to pull data out of the instructions variable and update the HMTL with what is found there.

Notice also that it mentions 2 additional functions that do not exist. getItem and updateElement. Your first task is to write those two functions. Then create another function to initialize the game and load the first instruction into the HTML and have that function called when the page loads.

The second part of this activity will be to finish the game. We need some way to know if we have reached an endpoint...either good or bad. Modify the JSON to include this information, then add a condition that checks each time we load a new step whether we have reached an endpoint or not.

If we have reached an endpoint it should then determine if we won or lost...and let us know.

Finally we should add a button that will reset the story.

Work together with your learning groups to finish this task. Include the names of those you worked with in a comment in your code.

Grading

Objective 5: Javascript

  • getItem function created that will pull a specified instruction from the list. (3pts)
  • updateElement function created that will update a specified element in the DOM (3pts)
  • Logic added to determine when the story ends. (4pts)

Objective 6: Content

  • The names of the students you collaborated with are listed in a comment in the JS. (5pts)
